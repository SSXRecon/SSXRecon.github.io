Accio

  • Type: Force
  • Description: Pulls objects towards user

Alohomora

  • Type: Essential
  • Description: Opens locked doors and chests

Arresto Mommentum

  • Type: Control
  • Description: Slows Object or enemy for a short amount of time

Avada Kedavra

  • Type: Unforgivable Curse
  • Description: Kills opponent instantly

Bombarda

  • Type: Damage
  • Description: Creates Explosion

Confrigo

  • Type: Damage
  • Description: Sends bolt of flames towards target

Crucio

  • Type: Unforgivable Curse
  • Description: Damages and pains enemy incapacitating them for a short time

Depulso

  • Type: Force
  • Description: Pushes target away from user

Descendo

  • Type: Force
  • Description: Slams target down into the floor

Diffindo

  • Type: Damage
  • Description: Long range attack

Disillusionment

  • Type: Utility
  • Description: Makes user less visible to enemies

Expelliarmus

  • Type: Damage
  • Description: Knocks wand or weapon from target

Flipendo

  • Type: Force
  • Description: Flips target

Glascius

  • Type: Control
  • Description: Freezes target

Imperio

  • Type: Unforgivable Curse
  • Description: Controls enemy for short amount of time

Incendio

  • Type: Damage
  • Description: Short-range fire attack

Levioso

  • Type: Control
  • Description: Levitates object

Lumos

  • Type: Utility
  • Description: Produces light from end of wand

Protego

  • Type: Essential
  • Description: Shields user from attacks

Reparo

  • Type: Utility
  • Description: Repairs broken objects

Revelio

  • Type: Essential
  • Description: Reveals hidden objects

Stupefy

  • Type: Essential
  • Description: Stuns enemies
